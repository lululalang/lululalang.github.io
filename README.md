# Kimlang THAN — Personal Portfolio Website ✨🌸

Welcome to the personal portfolio website for **Than Kimlang** (Digital Marketer & Content Specialist), hosted live on GitHub Pages.

## 🌟 Live Website
Visit: [https://lululalang.github.io](https://lululalang.github.io)

## 🎨 Design & Aesthetic Vibe
- **Girly & Chic Aesthetic**: Designed with soft pastel pinks, rose quartz accents, glassmorphism cards, Polaroid photo frames, and subtle sparkle animations.
- **Responsive Layout**: Optimized for mobile phones, tablets, and desktop devices.
- **Interactive Features**: Lightbox photo gallery, metrics counters, 1-click contact clipboard copy, and smooth navigation scrolling.

## 📁 Repository Structure
```
lululalang.github.io/
├── index.html               # Main HTML portfolio structure
├── css/
│   └── styles.css           # Custom aesthetic design system & layout styles
├── js/
│   └── main.js              # Interactive scripts (lightbox, copy button, scroll effects)
├── assets/

└── Kimlang Than's Portfolio.pdf  # Source resume/portfolio PDF
```

## 🛠️ Tech Stack
- **HTML5 & CSS3**: Vanilla CSS with custom properties, glassmorphism, and keyframe animations.
- **JavaScript (ES6+)**: Zero external JS dependencies for high performance.
- **FontAwesome 6 & Google Fonts**: *Playfair Display*, *Plus Jakarta Sans*, and *Caveat*.
- **GitHub Pages**: Fast, free static site hosting.
